* [Generic and package manager advice for caching](https://docs.travis-ci.com/user/caching/)

```yaml
language: python    
cache: pip
```

You can also manually specify which directories will be cached, though you can't 
choose the cache key.

```yaml
cache:
  directories:
    - $HOME/.cache/pip
before_cache:
  - rm -f $HOME/.cache/pip/log/debug.log
```
